תיאור המערכת ואופן פעולתה
מערכת זו מהווה אב-טיפוס יציב ומודולרי לבקרת כניסה ואבטחה היקפית במסגרת פרויקט "בית חכם". המערכת מבוססת על מיקרו-בקר ESP32 ומתוכנתת בשפת MicroPython (בסביבת Thonny), תוך יישום עקרונות תכנות מונחה-עצמים (OOP) המפרידים את שכבת החומרה מלוגיקת הניהול הראשית.
רכיבי החומרה ומנגנוני הגילוי
המערכת משלבת שתי שכבות הגנה משלימות הפועלות במקביל:
חיישן קטיעת קרן (Beam/Slot Sensor): משמש כמחסום אופטי מדויק ומזהה חסימה פיזית מיידית בקו הראייה שבין המשדר למקלט.
חיישן מרחק אולטרסוני (Ultrasonic Sensor): מודד נפח ומרחק באמצעות גלי קול, ומגדיר חדירה כאשר עצם מתקרב לטווח של פחות מ-20 ס"מ מהחיישן.

אופן פעולת התוכנה והלוגיקה
המערכת מריצה לולאת בקרה מהירה המנטרת את שני החיישנים ומפעילה שלושה מנגנוני עיבוד חכמים:
סינון רעשים וחסינות (Debounce Filter): חיישנים אולטרסוניים נוטים לסבול מרעשים וקריאות קצה שגויות עקב החזרי קול מגוף החודר. כדי למנוע ספירות כפולות והתראות שווא, שולב אלגוריתם סינון (Debounce). המערכת משנה את סטטוס חיישן המרחק רק לאחר קבלת מספר קריאות חריגות רציפות ועקביות.
ניהול מצבים (State Transition): הקוד עוקב דינמית אחר המצב הקודם של כל חיישן לעומת מצבו הנוכחי. הלוגיקה מזהה באופן בלעדי את רגע השינוי (המעבר מסטטוס תקין לסטטוס חדירה). מנגנון זה מבטיח כי גם אם עצם חוסם את החיישן לאורך זמן, המערכת תרשום חדירה בודדת בלבד ולא תבצע ספירה חוזרת של אותו אירוע.
דיווח ממוקד וניהול מונים: המערכת מנהלת מוני חדירה נפרדים ועצמאיים לכל רכיב. ברגע הזיהוי, המערכת מפענחת ומדפיסה בזמן אמת הודעה המציינת את מקור החדירה המדויק ([LASER SENSOR] או [DISTANCE SENSOR]) לצד הנתון המצטבר העדכני של אותו החיישן.
